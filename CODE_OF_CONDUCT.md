# Contributor Code of Conduct

## Our Commitment
We are committed to fostering an open, welcoming, and productive environment for everyone. All participants—maintainers, contributors, and users—are expected to uphold this standard.

## Expected Behavior
- Demonstrate respect and empathy toward others.
- Provide constructive feedback.
- Accept responsibility for your actions.
- Collaborate with professionalism.

## Unacceptable Behavior
- Harassment, discrimination, or personal attacks.
- Disruptive or unproductive behavior.
- Sharing private information without consent.

## Reporting Issues
If you observe behavior that violates this Code of Conduct, please contact the repository maintainers. Reports will be handled discreetly and respectfully.

## Enforcement
Maintainers may take appropriate action, including warnings or removal from the project, when the Code of Conduct is violated.

---

This Code of Conduct is adapted from widely used open‑source community guidelines.
